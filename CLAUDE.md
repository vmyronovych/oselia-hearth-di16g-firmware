## Agent skills

### Issue tracker

Issues are tracked in this repo's **GitHub Issues** via the `gh` CLI. External PRs are **not** a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default vocabulary: `needs-triage` / `needs-info` / `ready-for-agent` / `ready-for-human` / `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

**Multi-context** layout: `CONTEXT-MAP.md` at the root points to per-context `CONTEXT.md` files (`firmware/`, `homeassistant/`, `provisioning/`). See `docs/agents/domain.md`.
